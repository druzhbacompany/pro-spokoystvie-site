## 3. КАК ОБРЕЗАНЫ ЗАПРЕЩЁННЫЕ ЗОНЫ

Исходный баннер: 1080×1920. Композиция сверху вниз:
- логотип + «медицинский центр» + «ПРО спокойствие» + «и здоровье» (верх, ~0–850px) — **разрешено**;
- «Медицинские анализы HELIX», список специалистов, «IV-терапия, детокс-капельницы», «Кодирование от алкогольной зависимости», телефон-плашка (низ, ~890–1920px) — **запрещено**.

Обрезка выполнена через Python/Pillow (top-anchored crop):
```
img.crop((0, 0, 1080, 850))  →  about-banner-safe.jpeg (1080×850)
```
Срез на высоте **850px** отсекает всю нижнюю часть с HELIX, детокс-капельницами и кодированием, оставляя только верхний брендовый блок. Между «и здоровье» (заканчивается ~800px) и линией HELIX (начинается ~890px) есть чистый отступ — текст «и здоровье» сохранён полностью, HELIX и ниже **не попадают** в кадр (подтверждено визуальным просмотром обрезанного файла).
